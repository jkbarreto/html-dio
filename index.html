História da Web Primeiros Computadores

Os computadores surgiram no início com a principal função de ser usado para cálculos e eram bem maiores também, e com o decorrer dos anos os computadores começaram a ficar menores e com mais capacidade. Um desses computadores se chama ENIAC. O WWW foi criado para que todos pudessem ter acesso às informações e os códigos precisam estar sempre de acordo com os protocolos. A Microsoft teve uma grande sacada onde para popularizar os Browsers ( Navegadores ) começou a colocar em todos o scomputadores, tendo os mesmos o Navegador já instalado ao sair da fábrica.

Clients

Os clients são os clientes dos fornecedores, sao, são quem solicitam os dados dentro da internet, somos todos nós que acessamos as páginas, pois cada acesso necessitamos de uma resposta para termos o que precisamos.

Servers

Os Servers são nossos fornecedores de dados, que nos fornece as informações que aparecem para nós nos navegadores ou sites de compras ou qualquer site que exija login por exemplo, onde precisamos que eles retornem com a verificação se os dados que colocamos estão corretos e liberem o acesso para o que precisamos.

Linguagem de programação

Me surpreendi ao saber que HTML não é uma linguagem de programação, mas a linguagem permite uma conversa entre o Client e o Server, com isso, temos a base de dados e a base de frontend que seria o que visualizamos. É o que passa na internet para o que acontecer.

Não precisamos aprender todas as linguagens, precisamos pegar o rítimo e ter um obejtivo e aprender a linguagem que é sobre aquilo. E depois vamos nos aperfeiçoando. Pois tem linguagens que não trabalham juntas.

Introdução ao HTML

O HTML é formado por várias tags e códigos, e dentre eles segue alguns exemplos

  • Inspetor de Elementos

    • Nessa ferramenta podemos inspecionar os elementos, estudar e ver a estrutura do código

  • Tags

    • As tags são o que identifica para o computador a característica do texto digitado. Dá uma função e um noma para aquele texto.

  • Atributos

    • Os atributos servem para nomear tags para que possamos identificálos mais rápido e assim usar para links, cores e caracteríticas únicas.

  • Listas

    • Temos duas listas, a lista ordenada e a lista desordenada, onde na ordenada é colocado a númeração na sequencia para identificar aordem, e já na desordenada, são bolinhas sem uma ordem definida.
